# MAAP
Tienda de Ropa Elegante

**Problemática**

En la tienda de ropa MAAP, el registro de las facturas se realiza actualmente de forma manual, lo que genera errores en los datos, pérdida de información y dificultades para mantener un control adecuado de las ventas. Este procedimiento también complica la búsqueda de facturas por número o fecha y el cálculo correcto de los totales, especialmente cuando el volumen de transacciones aumenta. Como consecuencia, la gestión de ventas se vuelve lenta, poco confiable y propensa a fallos humanos, afectando la organización y el control administrativo del negocio.

**Objetivo del sistema**

El objetivo principal del sistema es desarrollar una aplicación con un menú de opciones que permita registrar y gestionar eficientemente las facturas de la tienda de ropa MAAP, automatizando el proceso de facturación. La aplicación permitirá registrar facturas, calcular automáticamente los totales, consultar información de ventas y visualizar un registro general, facilitando el acceso a la información y mejorando el control de las ventas mediante un sistema organizado, rápido y seguro.

**Requerimientos funcionales**

La aplicación de la tienda de ropa debe cumplir con las siguientes funcionalidades principales:

* Permitir el registro de productos mediante formularios gráficos.

* Ingresar datos como nombre del producto, tipo de prenda, talla, precio y cantidad.

* Validar que los campos obligatorios no estén vacíos.

* Validar que los datos ingresados sean correctos (por ejemplo, valores numéricos).

* Almacenar la información utilizando estructuras de datos (arreglos, listas o mapas).

* Mostrar mensajes de confirmación o error según la acción realizada.

* Permitir la visualización de los productos registrados.

* Asegurar que todas las funcionalidades estén alineadas al flujo definido del sistema.

**Conclución**

El proyecto MAAP – Tienda de Ropa permite aplicar los conocimientos de programación y acceso a datos mediante el uso de formularios gráficos y estructuras de datos.
La aplicación contribuye a mejorar la organización y el control de los productos, ofreciendo una solución funcional y eficiente para la gestión básica de una tienda de ropa.
